Logo-asset-specific color values (outside the token system):
The official logo SVG files contain two hex values not present in PI-Colour-Palette.csv:
#252e73 — dark wordmark color in PI_logo_hor-dark.svg. Closest CSV equivalent: Electric Indigo #2E4292. Do NOT use #252e73 as a design token outside the logo SVG.
#b8deff — light element in logo variants. Closest CSV equivalent: Baby Blue #81D1F4. Do NOT use #b8deff as a design token outside the logo SVG.
These values are pre-existing logo asset values that predate the CSV. The SVG files are the authoritative source for logo reproduction.
